This folder contains all the scripts for replication of paper-

"Disaster Exposure as an Instrument: Evaluating Income Effects on Education in Nepal"
Overleaf view link: Please email me for draft

The folder and files are maintained as follows:

1_data		- directory has data (removed for sharing purposes)
2_prep		- scripts for data cleaning
3_analysis 	- scripts for building figures and tables
4_log		- contains script log files (removed for sharing purposes)
5_fig		- contains figures built from scripts in 3_analysis
6_tab		- contains tables built from scripts in 3_analysis
8_documentation	- Documentation for data, survey questions, etc.

0_master.do	- Master/Control do file (open the file for more details!)
final_draft	- current final version of the paper

Notes on the Overleaf integration:
. Current version does not have automatic integration.
. Files from "5_fig" and "6_tab" are manually copied to
overleaf directory "figures" and "tables" respectively.

Notes on archiving:
. All directories have _archive directory within.
. Old versions of files should be compiled into a directory
before archiving.
. This directory should be named in following convention:
	. yymmdd"YMD"
	. For instance a directory named 260101YMD
	contains materials archived on January 1, 2026.
